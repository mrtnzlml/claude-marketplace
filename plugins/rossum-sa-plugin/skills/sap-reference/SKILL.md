# SAP Integration Reference

This skill provides a guide for integrating Rossum with SAP systems. It covers the SAP product landscape, integration patterns per SAP variant, master data exchange challenges, and real-world implementation examples. For complete details, see [reference.md](reference.md).

Use this knowledge when:
- Scoping a Rossum-SAP integration (understanding which SAP product the customer uses)
- Choosing an integration pattern (direct API, IDOC via middleware, SFTP, VIM API)
- Understanding SAP terminology (FICO, MIRO, GRNIV, IDOC, BAPI, etc.)
- Planning master data exchange with SAP systems
- Writing SOWs that involve SAP integration
- Reviewing or building export hooks that generate SAP-compatible output (INVOIC02, ORDERS05)
- Understanding middleware requirements for S4 HANA Private Cloud / ECC integrations
